# My Custom New Tab v1.0.0

A modern and customizable Chrome New Tab dashboard built with HTML, CSS, and JavaScript.

## Features

* Live Clock & Date
* Search Bar with Multiple Search Engines

  * Google
  * DuckDuckGo
  * YouTube
* Quick Access Links
* Notes with Local Storage
* Motivational Quotes
* Responsive Design
* Lightweight and Fast

## Preview

A clean dashboard that replaces Chrome's default New Tab page with:

* Welcome section
* Search section
* Quick Links
* Notes area
* Daily motivational quote

## Installation

### Method 1: Download from GitHub

1. Download or clone this repository.

2. Open Chrome and navigate to:

   chrome://extensions

3. Enable **Developer Mode**.

4. Click **Load unpacked**.

5. Select the project folder.

6. Open a new tab to see the dashboard.

## Project Structure

My-Custom-New-Tab/

├── manifest.json      # Chrome extension configuration

├── index.html         # Main dashboard page

├── style.css          # Dashboard styling

├── script.js          # Dashboard functionality

├── README.md          # Project documentation

├── LICENSE            # MIT License

└── .gitignore         # Git ignore rules

## Files Overview

* **manifest.json** — Defines the Chrome extension and overrides the New Tab page.
* **index.html** — Contains the dashboard layout and content.
* **style.css** — Controls the appearance and design of the dashboard.
* **script.js** — Handles the clock, search, notes, and quotes functionality.
* **README.md** — Documentation and installation instructions.
* **LICENSE** — Project license information.
* **.gitignore** — Files and folders excluded from Git tracking.
